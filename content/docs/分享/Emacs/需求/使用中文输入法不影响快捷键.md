## Rime-安装 {#rime-安装}

```elisp
(use-package rime
  :custom
  (rime-librime-root my/rime-librime-root)
  (default-input-method "rime")
  (rime-user-data-dir my/rime-user-data-dir)
  (rime-show-candidate 'posframe)
  :bind
  (:map rime-mode-map
        ("C-`" . 'rime-send-keybinding)))
```


## 设置符号半角/全角 {#设置符号半角-全角}

1.  C-\` <br/>
2.  选择 `。，->.,` <br/>
