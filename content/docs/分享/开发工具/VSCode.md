## 快捷键 {#快捷键}

| 操作       |                     |
|----------|---------------------|
| 减小字体   | Command--           |
| 放大字体   | Command-+           |
| 文件内查找 | Command-F           |
| 注释代码   | Command-K Command-C |
| 去注释     | Command-K Command-U |
| 头文件/源文件 切换 | Option-O            |
| 打开首选项 | Command-,           |


## 使用clang-format {#使用clang-format}

1.  安装扩展C/C++，自动安装clang-format <br/>
2.  保存文件时，应用格式 <br/>
    `首选项 > Text Editor > Formatting，勾选 Format On Save` <br/>
