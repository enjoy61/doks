## 键盘操作 {#键盘操作}

可以在 `系统偏好设置 > 键盘 > 快捷键 > 调度中心` 中设置快捷键 <br/>

|              | -                       |                              |
|--------------|-------------------------|------------------------------|
| Control-Up   | 进入调度中心：查看当前桌面内应用程序和桌面选项 | 按Control-Up / Control-Down返回 |
| Control-Down | 列出当前应用的所有窗口  | 同上                         |

|                              | -       |
|------------------------------|---------|
| Control-Left / Control-Right | 左右切换桌面 |
| Control-NUM                  | 切换到指定桌面 |


## 触控板操作 {#触控板操作}

可以在 `系统偏好设置 > 触控板 > 更多手势` 中进行设置 <br/>

|         | -           |
|---------|-------------|
| 三指 向上轻扫 | 进入调度中心 |
| 三指 向下轻扫 | 列出当前应用的所有窗口 |
| 三指 左右轻扫 | 左右切换桌面 |
